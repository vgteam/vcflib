# NAME

**vcfaltcount**

# SYNOPSIS

**vcfaltcount** <vcf file>

# DESCRIPTION

count the number of alternate alleles in all records in the vcf file



# OPTIONS

```


Type: statistics

```





# EXIT VALUES

**0**
: Success

**not 0**
: Failure
